# Setup e Execução - Product Service

---

## Requisitos

- Java 17+
- Spring Boot
- JPA (Hibernate)
- Flyway
- Lombok

---

## Dependências principais

- `spring-boot-starter-data-jpa`
- `spring-boot-starter-web`
- `flyway-core`
- `lombok`

---

## Como compilar

```bash
mvn clean package
```

---

## Validações

- Nome do produto é obrigatório

- Preço deve ser positivo
